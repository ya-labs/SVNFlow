# RFCs

## Objetivo

Esta pasta guarda propostas ainda não aceitas como decisão.

Use RFC quando uma ideia precisar ser discutida antes de virar ADR, requisito ou implementação.

## Diferença entre RFC e ADR

| Tipo | Uso |
| --- | --- |
| RFC | Proposta em discussão. |
| ADR | Decisão aceita, com contexto e consequências. |

## Exemplos de temas futuros

- integração com GitHub;
- modo colaborativo;
- suporte a `git svn`;
- automações adicionais;
- suporte ampliado a arquivos binários.

Não registre RFC como decisão final.
